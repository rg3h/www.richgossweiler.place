QuickSuggest: Character Prediction for Improved Text Entry on Web Appliances

Ullas Gargi, Rich Gossweiler
While research efforts are under way to improve the traditional remote control, the simple left-right-up-down remote currently dominants as the TV input device. But usiing it to navigate an onscreen keyboard is painful. Can we use a Google Suggest model to make it easier?
This model creates a highlight "ring" of predicted characters around the currently selected character. Thus the onscreen layout is not disturbed, but entering the next character may only be one click away.
